Web Service Interface











Request method for retrieving information on all teams
GET
Resource path for retrieving information on all teams
https://student.csc.liv.ac.uk/~sgnsee/v1/teams
Request method for retrieving information on all players of team {teamId}
GET
Resource path for retrieving information on all players of team {teamId}
https://student.csc.liv.ac.uk/~sgnsee/v1/teams/{teamId}/players
Request method for retrieving information on player {playerId} of team {teamId}
GET
Resource path for retrieving information on player {playerId} of team {teamId}
https://student.csc.liv.ac.uk/~sgnsee/v1/teams/{teamId}/players/{playerId}
Request method for adding a player to team {teamId}
POST
Resource path for adding a player to team {teamId}
https://student.csc.liv.ac.uk/~sgnsee/v1/teams/{teamId}/players
Sample request body for adding a player to team {teamId}
{
     "surname": "John",
     "givenName": "Smith",
     "nationality": "British",
     "dob": "2000-01-31"
}
Request method for updating parts of the information on player {playerId} of team {teamId}
PATCH
Resource path for updating parts of the information on player {playerId} of team {teamId}
https://student.csc.liv.ac.uk/~sgnsee/v1/teams/{teamId}/players/{playerId}
Sample request body for updating parts of the information on player {playerId} of team {teamId}
{
     "surname": "Jason",
     "givenName": "Borne",
     "nationality": "American",
     "dob": "2000-02-01"
}
Request method for deleting player {playerId} from team {teamId}
DELETE
Resource path for deleting player {playerId} from team {teamId}
https://student.csc.liv.ac.uk/~sgnsee/v1/teams/{teamId}/players/{playerId}
